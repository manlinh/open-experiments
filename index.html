Project NP

Rational Reconstruction & $p$-adic Lifting for LWE Cryptanalysis

NTNU Department of Computer Science ID: 40947044S 40947044S@ntnu.edu.tw

Abstract

Project NP introduces a deterministic framework for resolving Learning With Errors (LWE) secrets by treating noise as periodic 2-adic rational fractions. Unlike traditional lattice reduction, this engine utilizes a recursive parity-lifting formula to "bridge" the gap between Truth ($P$) and Falsehood ($N$). We demonstrate that information persistence allows for a NAND-style spark even from a vacuum initialization ($0,0$).

The Mathematical Core

The Universal Inference Formula:

$$NP_{s+1} = \frac{NP_s + P_s + N_s}{2^{\text{v}_2(P_s \oplus N_s)}}$$

Kummer Lifting ($v_2$)

Uses 2-adic valuation to "zoom in" on the first bit of disagreement between truth and noise.

The Parity Bridge

A successful inference is confirmed when:
$$NP \pmod N = NP \pmod P$$

Key Insights

  • ✔
    The "Haha" Moment: Rational Origin

    LWE noise is effectively a "faked" rational fraction. By treating it as such, we can identify periodic bit-patterns (like $1/3$ in $\mathbb{Q}_2$) to reconstruct the secret.

  • ✔
    Symmetry Breaking

    For the inference to "breathe," $N$ and $P$ must be different. This forces a choice between parallel universes that the engine eventually resolves.

  • ✔
    Vacuum Initialization

    Information is never truly lost. Even starting from absolute zero, the interaction of carries produces a logic "spark."

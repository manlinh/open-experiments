核心洞察 (The Insight)

傳統 LWE 攻擊依賴複雜的格點矩陣（Lattice Matrix）縮減，但在 Project NP 中，我們將雜訊視為一個 「尚未位移的進位 (Unshifted Carry)」。

這不是線性代數問題，而是數論中的 有理數重構。就像在 $2$-adic 中解開 $1/3$ 的循環規律，秘密就藏在除法產生的餘數序列裡。

v2

// NP Inference Formula

$$NP_{s+1} = \frac{NP_s + P_s + N_s}{2^{\text{v}_2(P_s \oplus N_s)}}$$

透過 $2^{v_2}$ 提升位元深度，精確定位雜訊。

邊界分析 (Boundary Analysis)

Modular Cliff (模數懸崖)

當 Truth ($P$) 的增長觸及模數 $Q$ 時，位元邏輯會發生非線性翻轉。如何處理位元運算與同餘運算的不相容性是目前的關鍵挑戰。

Error Propagation (誤差累積)

$p$-adic 提升對 LSB 極度敏感。若低位元的推論出現 1-bit 偏差，會導致高位元序列完全偏離正確的有理路徑。

* 本專案目前針對純量 (Scalar) LWE 進行驗證，高維矩陣擴展仍在實驗中。

核心洞察 (The Insight)

傳統 LWE 攻擊依賴複雜的格點矩陣（Lattice Matrix）縮減，但在 Project NP 中，我們將雜訊視為一個 「尚未位移的進位 (Unshifted Carry)」。

這不是線性代數問題，而是數論中的 有理數重構。就像在 $2$-adic 中解開 $1/3$ 的循環規律，秘密就藏在除法產生的餘數序列裡。

v2

// NP Inference Formula

$$NP_{s+1} = \frac{NP_s + P_s + N_s}{2^{\text{v}_2(P_s \oplus N_s)}}$$

透過 $2^{v_2}$ 提升位元深度，在不使用矩陣的情況下精確定位雜訊。

Division Over Lattice

捨棄高維矩陣運算，改採有理數長除法的遞迴邏輯，處理 LWE 的尾數誤差。

Symmetry Breaking

強制 Truth ($P$) 與 False ($N$) 異值，利用 XOR 距離誘導系統從真空產生推論。

Hensel Lifting

從 LSB (最低有效位) 逐個位元「提升」出解，無視 LWE 的模糊初始狀態。
